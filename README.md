## Project Setup

```sh
npm install
```

## Initialize Husky

```sh
npx husky init
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Format with [Prettier](https://prettier.io/)

```sh
npm run format
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
